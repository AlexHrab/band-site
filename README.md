# Band Site

Лендінг-сайт українського рок-гурту "Грим та Грім" з секціями про гурт, концерти, учасників, контактами та формою бронювання.

## Технології

- Vite
- Vanilla JavaScript (ES Modules)
- HTML partials
- CSS

## Можливості

- Адаптивний інтерфейс з мобільним меню
- Таблиця концертів з пагінацією ("Показати ще")
- Бронювання квитків через модальне вікно
- Збереження стану бронювань у `localStorage`
- Форма зворотного зв'язку
- Базові SEO та accessibility покращення

## Оптимізації продуктивності

- Зображення підключені через `AVIF/WebP` з fallback на `JPG`
- Для головного фону секції hero використано `image-set(...)`
- Додано preload для hero-зображення в `index.html`
- Для не-критичних зображень увімкнено `loading="lazy"` і `decoding="async"`
- Рендер таблиці концертів оптимізовано через `DocumentFragment`
- Для кнопок у таблиці використано делегування подій (менше listeners)
- Додані guard-перевірки ініціалізації, щоб уникати дублювання обробників
- Спільне сховище стану концертів (`concerts-store`) з швидким доступом по `id`

## Структура проєкту

- `index.html` - основна сторінка та підключення partials
- `src/main.js` - ініціалізація частин сторінки та модулів
- `src/partials/` - HTML-секції сторінки
- `src/js/` - логіка інтерфейсу
- `src/css/` - стилі
- `src/assets/` - зображення та дані концертів
- `public/` - публічні файли (`robots.txt`, `sitemap.xml`)

## Швидкий старт

1. Встановити залежності:

```bash
npm install
```

2. Запустити dev-сервер:

```bash
npm run dev
```

3. Зібрати production-версію:

```bash
npm run build
```

4. Переглянути production-збірку локально:

```bash
npm run preview
```

## Нотатки

- Перед деплоєм перевір домен у `public/robots.txt` і `public/sitemap.xml`.
- Стан бронювань зберігається у браузері користувача (`localStorage`).
